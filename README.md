# 🚀 Job Application Form

## 📌 Description
A simple online job application form built using HTML and CSS.  
This project demonstrates form structure, input fields, and basic UI design.

---

## ✨ Features
- Personal Information Form  
- Education Details Section  
- Work Experience Section  
- Radio Buttons and Checkboxes  
- Clean and Simple UI  

---

## 🛠 Technologies Used
- HTML5  
- CSS3
